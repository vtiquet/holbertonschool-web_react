# Holberton School Web Stack - React

This repository contains projects and exercises related to the Web Stack specialization at Holberton School, with a focus on **React** and **TypeScript**.

## Project Overview

The journey begins with a deep dive into **TypeScript** to ensure type safety, better developer experience, and robust code architecture.

## Directories

| Directory | Description | Key Concepts |
| :--- | :--- | :--- |
| [**TypeScript**](./TypeScript) | Foundations of TS | Interfaces, Classes, Namespaces, Nominal Typing |

## Learning Objectives
* Implement complex data structures using TypeScript **Interfaces**.
* Use **Namespaces** and **Declaration Merging** to organize large codebases.
* Understand the difference between **Structural** and **Nominal** typing.
* Build and bundle modern web applications using **Webpack**.
